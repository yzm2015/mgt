## 2. 产品概述

### 2.1 产品定位

《磁吸拆迁队》是一款基于物理模拟的休闲消除类微信小游戏。玩家操控起重机的磁铁吸附建筑方块，利用钟摆物理惯性抛掷方块撞击建筑，在限定时间内达到目标分数即可过关。

### 2.2 目标用户

| 维度 | 描述 |
|------|------|
| 年龄 | 12-45岁 |
| 性别 | 不限 |
| 平台 | 微信小游戏（iOS + Android） |
| 场景 | 碎片时间消遣、解压放松 |
| 画像 | 喜欢物理模拟/策略消除/休闲游戏的微信用户 |

### 2.3 核心体验

**"一吸一放，建筑崩塌"** — 极简操作下的物理破坏快感。

### 3.2 操作系统

#### FR-3.2.1 触摸控制

| 编号 | 操作 | 触发区域 | 效果 |
|------|------|----------|------|
| FR-3.2.1.1 | 滑动 | 屏幕左下1/3区域 | 起重机左右移动，方向和距离跟随手指 |
| FR-3.2.1.2 | 长按 | 屏幕中上2/3区域 | 激活磁铁，自动吸附范围内最近方块 |
| FR-3.2.1.3 | 释放 | 任意位置 | 释放吸附的方块，给予基于钟摆角速度的抛掷力 |
| FR-3.2.1.4 | 点击 | UI 按钮区域 | 触发对应按钮功能 |

**触摸坐标转换规则**：
- 原始坐标：`e.touches[0].clientX / clientY`
- 转换公式：`canvasX = touchX - safeAreaLeft`，`canvasY = touchY - safeAreaTop`
- DPI 适配：逻辑坐标 = 物理坐标 / pixelRatio

## 5. 数据结构

### 5.1 游戏状态

```javascript
gameState = {
  score: number,           // 当前分数
  timeLeft: number,        // 剩余时间（秒）
  gameActive: boolean,     // 游戏是否活跃（控制玩法逻辑）
  gamePaused: boolean,     // 是否暂停
  currentLevel: number,    // 当前关卡
  targetScore: number,     // 目标分数
  stars: number,           // 获得星级
  currentScene: string,    // 当前场景：menu/game/levelselect/leaderboard/instructions/gameover
  unlockedLevel: number    // 已解锁的最高关卡
}
```

### 5.2 起重机数据

```javascript
crane = {
  x: number,               // 中心 X 坐标
  y: number,               // 顶部 Y 坐标
  width: number,           // 宽度

magnet = {
  x: number,               // 中心 X 坐标
  y: number,               // 中心 Y 坐标
```
