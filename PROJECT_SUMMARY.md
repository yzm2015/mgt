## 🎮 游戏操作说明

### 微信小游戏版
1. **左下区域滑动**：控制起重机左右移动
2. **长按屏幕**：激活磁铁，吸附附近物体
3. **松开手指**：释放物体，利用惯性撞击建筑
4. **点击右上角按钮**：暂停/继续游戏

### 浏览器测试版
1. **鼠标左键拖动**：控制起重机左右移动
2. **鼠标右键点击**：激活磁铁
3. **释放右键**：释放物体
4. **空格键**：暂停/继续
5. **R 键**：重新开始
6. **N 键**：下一关

## 🚀 如何运行

### 方法一：浏览器测试（推荐）

1. 打开 `test.html` 文件（双击或在浏览器中打开）
2. 游戏将自动加载并运行
3. 使用鼠标和键盘进行测试

**注意事项**：
- 浏览器版仅用于逻辑测试
- 部分微信 API（如震动）会显示为 console.log
- 音效需要在 `audio/` 目录添加音频文件

### 方法二：微信开发者工具

1. 下载并安装微信开发者工具
2. 选择「小游戏」项目类型
3. 导入 `magnetic-demolition` 文件夹
4. 使用测试号或填入 AppID
5. 点击「编译」运行

## 🔧 扩展开发指南

### 添加新关卡

编辑 `js/BuildingGenerator.js` 的 `getLevelConfig` 方法：

```javascript
4: {
  timeLimit: 90,
  targetScore: 1500,
  blocks: [
    { x: 100, y: 300, type: 'steel', width: 60, height: 60 },
    // 添加更多方块...
  ]
}
```

### 调整游戏难度

编辑 `game.js` 的 `CONFIG` 对象：

```javascript
const CONFIG = {
  gravity: 0.5,           // 重力系数（建议 0.3-0.8）
  ropeLength: 150,         // 绳索长度（越长越难控制）
  magnetRadius: 80,        // 磁铁吸附范围
  craneSpeed: 5             // 起重机移动速度
};
```

### 添加音效

1. 将音频文件放入 `audio/` 目录
2. 在 `game.js` 中加载音效：

```javascript
audioManager.loadSound('grab', 'audio/grab.mp3');
audioManager.loadSound('release', 'audio/release.mp3');
```

3. 在相应位置播放音效：

```javascript
audioManager.playSound(SoundNames.GRAB);
```
